# Calculator-gui
built with python using tkinter

## Features
- Addition, subtraction, multiplication, division
- Square root (√)
- Percentage (%)
- Toggle positive/negative (+/-)

## What I used
- Python
- tkinter (built-in Python library)

## How to run
Make sure you have Python installed, then run:
calculator_gui.py

## What I learned
- Building GUIs with tkinter
- Managing application state with global variables
- Dynamically creating widgets with loops
- Handling user input and edge cases (e.g. division by zero)
